## Spatial Analysis on Meuse Dataset

### Introduction of meuse dataset
The Meuse River dataset we used here is included in `sp` package in R. This data set gives locations and topsoil heavy metal concentrations, along with a number of soil and landscape variables at the observation locations, collected in a flood plain of the river Meuse, near the village of Stein (NL).
